.. title:: pyvista_qt


Getting Started
***************

Installation using ``pip`` is simply::

    $ pip install pyvista_qt

You can also visit `PyPi <https://pypi.org/project/pyvista/>`_ or
`GitHub <https://github.com/pyvista/pyvista>`_ to download the source.
